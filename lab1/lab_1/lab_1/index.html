REST-сервис на основе ASP.NET HTTP-обработчика

Endpoints:

  • GET: .kak — Получить RESULT (текущее значение RESULT + верхний элемент стека)
  • POST: .kak?result=... — Изменить значение RESULT (по умолчанию RESULT = 0)
  • PUT: .kak?add=... — Добавить значение в стек
  • DELETE: .kak — Удалить значение из стека
